# Auto Moiller

## 概要

超シンプルなオートフィラーを作ったよ。
シンプルすぎるがゆえに制約とバグも多いよ。
でもパスワードの悪用とかはしないよ。コード一瞬で見れるから確認してみてね。

## インストール

1. ローカルに落とす
1. chromeの拡張機能画面を開く
1. デベロッパーモードにする
1. 「パッケージ化されていない拡張機能を読み込む」からこのディレクトリを指定する

## 使い方

1. 拡張機能からピン止めする
1. アイコンを押す

Save: その画面に存在する input, textarea, select の内容を保存（ただし、name属性必須）

Input: 保存した内容を反映

Clear: 保存した内容を全てクリア
